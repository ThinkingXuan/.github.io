原因

每次写博客都要新建一个.md文件，然后把博客内存复制进去。然后执行hexo clean && hexo g && hexo d ，接着还要输入账号密码，这已经比较麻烦了，然后，你在博客预览的时候如果发现有些小错误，还要进行部署和输入密码一系列操作，很浪费时间。
