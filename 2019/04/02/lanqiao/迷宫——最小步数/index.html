迷宫—最小步数
发表于 2019-04-02| 更新于 2020-03-05|算法
阅读量:|评论数:

描述

这有一个迷宫，有0~8行和0~8列：

Code

0表示道路，1表示墙。

现在输入一个道路的坐标作为起点，再如输入一个道路的坐标作为终点，问最少走几步才能从起点到达终点？

（注：一步是指从一坐标点走到其上下左右相邻坐标点，如：从（3，1）到（4,1）。）

输入

第一行输入一个整数n（0<n<=100），表示有n组测试数据;
随后n行,每行有四个整数a,b,c,d（0<=a,b,c,d<=8)分别表示起点的行、列，终点的行、列。

输出

输出最少走几步。

样例输入

2

3 1 5 7

3 1 6 7

样例输出

12

11

解法一：DFS + 剪枝

c

解法二 :BFS

c
